# timewait
Simple console countdown
```
timewait v1.0.0.0
simple console countdown

Usage:
    timewait.exe t
Parameters:
    t   timeout interval in seconds
Example:
    timewait.exe 10     wait for 10 seconds
Remarks:
    <Enter>, <Space> proceed; <Esc> abort
    proceed - set errorlevel 0
    abort   - set errorlevel 1
```
